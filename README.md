# McLaren Website

A modern, responsive website showcasing the McLaren car brand with a sleek dark theme and smooth animations.

## Features

✨ **Modern Design**
- Dark theme with orange accent colors (McLaren brand colors)
- Smooth animations and transitions
- Responsive grid layouts

🚀 **Performance**
- Lightweight and fast-loading
- Optimized CSS animations
- Smooth scrolling interactions

📱 **Responsive**
- Mobile-friendly design
- Hamburger menu for small screens
- Adaptive grid layouts

## Sections

1. **Navigation Bar** - Fixed navbar with smooth scrolling links
2. **Hero Section** - Eye-catching landing area with animated background
3. **Models** - Showcase of three McLaren models (720S, GT, Artura)
4. **Features** - Key features highlighting McLaren's excellence
5. **Contact** - Contact form for inquiries
6. **Footer** - Footer with social links

## Files

- `index.html` - HTML structure
- `styles.css` - Styling and animations
- `script.js` - Interactive functionality

## How to Use

1. Download or clone this repository
2. Open `index.html` in your web browser
3. Customize content, colors, and images as needed

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #ff6b00;      /* Orange */
    --dark-bg: #0a0e27;             /* Dark blue */
    --text-light: #ffffff;          /* White */
    --text-gray: #b0b0b0;           /* Gray */
    --accent: #ff8c00;              /* Lighter orange */
}
```

### Content
- Update model names, descriptions, and prices in the Models section
- Modify feature icons and descriptions in the Features section
- Change company name and social links in the Footer
